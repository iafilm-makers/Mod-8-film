Camera_Sanky620.ino is the working code for camera "Prototype One" as at 13 May 2026.
This codes a single MCU, using a polling loop to "multitask" the running of 3 x stepper motors
and a laser light source to "zap" (expose) a registration mark on the film edge.

Camera_NEMA_Test01_self-advance.ino is simpler code for individual testing of stepper motors.
